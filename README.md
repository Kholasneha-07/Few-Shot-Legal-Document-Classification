# Few-Shot Legal Document Classification

## 🌐 Live Demo
https://leagldocumetclsifictin.lovable.app/

## 📌 Project Description
This project classifies legal documents using few-shot learning techniques. It is designed to work with minimal data and still provide meaningful predictions.

## 🎯 Objective
To build an AI-based system that can categorize legal text efficiently even with limited training data.

## 🛠️ Technologies Used
- Artificial Intelligence
- Machine Learning
- NLP Concepts
- Lovable (for deployment)

## 🚀 Features
- Classifies legal documents
- Works with small datasets
- Simple and user-friendly interface
